# 🏫 SBOA Admission System

AI-powered school admission portal for 13 SBOA schools in Tamil Nadu, powered by Groq AI.

## ⚠️ Setup — Add Your Groq API Key

1. Go to your Space → **Settings** tab
2. Scroll to **Repository secrets**
3. Click **New secret**
4. Name: `GROQ_API_KEY` | Value: your key from [console.groq.com](https://console.groq.com)
5. Click Save → **Factory reboot** the Space

## Features
- 💬 AI Chatbot (24/7, English / Hindi / Tamil)
- 📄 Multi-document OCR + Auto Form Fill
- ✅ Document Validation
- 🔍 Fraud + Duplicate Detection
- 🏆 Smart Applicant Scoring
- 📈 AI Demand Forecasting
- 🌐 All 13 schools Super Admin Dashboard
